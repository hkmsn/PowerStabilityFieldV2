Pro-Tip for VS Code
Since you are mixing Monkey C and PlatformIO (C++/C), VS Code might get confused with extensions.
I recommend using Multi-Root Workspaces. 
You can go to File > Add Folder to Workspace... 
and add the monkeyc_project and platformio_project folders individually. 
This allows the Monkey C extension and the PlatformIO extension
 to run only in their respective directories, preventing build tool conflicts.
